## Ege Özkoç

PhD student in Electrical & Computer Engineering at the University of Pittsburgh.
I'm working on machine learning and deep learning for biomedical signals and
healthcare, mainly ECG and EEG. I also work on explainable AI, such as using large
language models to translate model outputs into clear explanations that help
clinicians make decisions.

I like understanding systems by building them from the ground up, which is where
most of the projects below come from. I'm also interested in machine learning and
deep learning systems.

### Selected projects

- **[mini-numpy](https://github.com/egeozkoc/mini-numpy)** — a NumPy-like array
  library written in modern C++, with Python bindings via pybind11.
- **[deep-learning-from-scratch](https://github.com/egeozkoc/deep-learning-from-scratch)** —
  a neural-network framework built in NumPy: conv nets, RNNs/LSTMs, batchnorm,
  Adam, and gradient checking, with no autograd.
- **[lora-from-scratch](https://github.com/egeozkoc/lora-from-scratch)** — LoRA
  implemented directly in PyTorch: the low-rank adapter, injection into existing
  layers, and weight merging.
- **[murmur](https://github.com/egeozkoc/murmur)** — local, offline dictation for
  macOS powered by Whisper; a free, on-device alternative to Wispr Flow.
- **[clustering-toolbox](https://github.com/egeozkoc/clustering-toolbox)** — an
  interactive Streamlit app for clustering and exploring CSV datasets.
